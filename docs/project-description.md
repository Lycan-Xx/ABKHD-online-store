# ABKHD Store Project Overview

ABKHD Store is a modern, high-performance ecommerce platform purpose-built for tech enthusiasts operating in the Nigerian market. Developed with Astro 5 and Appwrite backend infrastructure, this open-source project delivers a premium shopping experience that combines static site generation speed with dynamic server-side capabilities.

## How It Works
This platform leverages Astro's hybrid rendering architecture to deliver lightning-fast static pages while maintaining dynamic functionality for cart operations, checkout workflows, and administrative functions. Appwrite provides all backend services including database management, secure file storage, and user authentication without requiring custom server code. The platform integrates the Squad payment gateway natively to support local Nigerian payment methods including cards, bank transfers, and USSD transactions.

## Relevance
ABKHD Store addresses critical gaps in African ecommerce solutions by providing a fully open-source, production-ready platform optimized for local payment infrastructure, regional delivery logistics, and mobile-first user experiences. Unlike generic international ecommerce platforms, it includes region-specific features that eliminate the need for extensive customization for Nigerian businesses.

## Latest Features (April 2026 Update)
The most recent release introduces a completely redesigned multi-step checkout flow that improves conversion rates by breaking the purchase process into logical stages. New delivery zone support with configurable fees for Yola and Jimeta areas enables accurate shipping calculations. Customers can now alternatively place orders directly through WhatsApp for manual processing. Technical improvements include proper stock management to prevent overselling, enhanced payment webhook reliability, and fixed Cloudflare Pages deployment configuration that resolves environment variable issues.

Additional standout features include product video previews with automatic thumbnail generation, robust upload caching that prevents media loss during network failures, bulk product management operations, system-aware dark mode, persistent shopping cart functionality, and a full-featured administrative dashboard for complete store management.
